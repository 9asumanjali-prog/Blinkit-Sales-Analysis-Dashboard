# Blinkit-Sales-Analysis-Dashboard
Sales Trends & Performance Dashboard
Sales Analytics : Business Insights Dashboard
The Blinkit Sales Analysis Dashboard is designed to simplify complex sales data using interactive visualizations.It enables users to explore business performance, compare outlet efficicency, and evaluate product categories, making data analytics easier,faster and more effective.

The dashboard was built using the following tools and technologies:
*📊Power BI Desktop - Main data visualization platform used for report creation.
*🔄 Power Query - Cleaned, transformed, and prepared raw data for accurate analysis.
*📐DAX (Data Analysis Expression)- Created calculated measures, KPIs, and custom formulas to generate meaningful insights.
*📈 Data Visualization - Used KPI cards, line charts, bar charts, donut charts, tables, and slicers to present data effectively.
*🎛️Slicers and Filters - Enabled interactive filtering by Outlet Location, Outlet Size, and Item Type for better exploration.
*🎨Dashboard Design – Applied a clean, user-friendly interface with Blinkit's signature yellow and green theme for an attractive and intuitive user experience.
*✨Business Intelligence – Transformed raw sales data into meaningful insights to support better decision-making and performance analysis.
*📂Excel Dataset (CSV/XLSX) – Used as the primary data source for the dashboard.

*Source: Blinkit Sales Dataset
The Blinkit Sales Dataset contains 8,523 records and 12 attributes, covering outlet establishment years from 2011 to 2022. It includes product details, outlet information, sales, and customer ratings used for dashboard analysis.

*Features/Highlits
💰 KPI Cards – Display Total Sales, Average Sales, Number of Items, and Average Rating.
🥗 Fat Content Analysis – Compares sales by Low Fat and Regular products.
🛒 Item Type Analysis – Shows sales contribution of different product categories.
🏪 Outlet Size & Location – Analyzes sales across outlet sizes and locations.
📅 Outlet Establishment Trend – Displays sales based on establishment year.
📊 Outlet Type Summary – Compares outlet types using sales, ratings, and item count.

*🎯 Business Problem  
Blinkit collects large amounts of sales data, making it difficult to monitor performance and identify trends. The dashboard solves this by converting raw data into interactive insights that help improve decision-making and business performance.

* Key Questions
💰 What is the total sales generated by Blinkit?
📦 How many items have been sold?
⭐ What is the average customer rating?
📈 What is the average sales per item?
🛍️ Which item categories contribute the most to sales?
🥗 How do Low Fat and Regular products compare in sales?
🏪 Which outlet type generates the highest revenue?
📍 Which outlet location tier performs the best?
📏 How does outlet size impact sales performance?
📅 How have sales changed based on outlet establishment year?

*💡 Business Insights
📊 Sales Performance: Total sales reached $1.20M.
🛍️ Top Products: Regular Fat products generated the highest sales.
🏪 Best Outlet: Supermarket Type 1 performed the best.
📍 Top Location: Tier 3 outlets achieved the highest revenue.
📦 Outlet Size: Medium-sized outlets contributed the most sales.
